Executando o projeto

O gerenciamento de dependências do projeto é feita através do Poetry; recomendamos que utilize esta ferramenta para realizar a instalação das bibliotecas utilizadas com facilidade. Caso deseje, as instruções para instalação e execução do projeto utilizando somente pip e venv estão disponíveis na próxima seção.

  1. Instale o Poetry para realizar a instalação das dependências (recomendado instalar usando o pipx, para não interferir com os ambientes virtuais do seu sistema operacional):
pipx install poetry
  1. Instale as dependências do projeto:
# pwd: INE5608-2024.02-4.0.0-entrega/src
poetry install
  1. Execute o projeto:
# pwd: INE5608-2024.02-4.0.0-entrega/src
poetry run alapo

Alternativamente, a instalação e execução pode ser feita diretamente via pip:

  1. Instancie e ative um ambiente virtual para a instalação do projeto:
# pwd: INE5608-2024.02-4.0.0-entrega/src
python -m venv .venv
source .venv/bin/activate
  1. Instale as dependências do projeto:
# pwd: INE5608-2024.02-4.0.0-entrega/src
pip install -r requirements.txt
  1. Execute o projeto:
# pwd: INE5608-2024.02-4.0.0-entrega/src
python -m alapo
